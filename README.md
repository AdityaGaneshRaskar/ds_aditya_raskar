# Trader Behavior & Market Sentiment Analysis

## Objective
Analyze how market sentiment (Fear vs Greed) impacts trader profitability, trade size, and activity.

## Repository Structure
- notebook_1.ipynb → Complete analysis (Google Colab)
- csv_files/ → Dataset
- outputs/ → Generated visualizations
- ds_report.pdf → Final report

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Note
Trade Size (USD) is used as a proxy for leverage due to unavailability of leverage data.
